# Ledgefy


Ledgefy is a lightweight budgeting app built to simplify personal finance tracking. It allows users to manage income, expenses, and view spending insights through a clean and intuitive interface. Designed with MVVM architecture and persistent storage for a smooth and scalable user experience.
